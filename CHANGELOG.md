## v0.1.0 - 2026-04-28

Initial public release.

- Provides a local Qt desktop assistant for Rock Kingdom PVP screenshot analysis
- Uses multimodal model APIs directly instead of local OCR
- Includes public pet/skill wiki datasets and SQLite knowledge database
- Supports local knowledge retrieval plus round-by-round battle advice
